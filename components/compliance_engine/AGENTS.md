# Compliance Engine Agent

## Role
This component enforces regulatory compliance for reptile keeping (CITES, I-FAP, local laws).

## Responsibilities
1. Valiate animal records against species requirements.
2. Alert on missing documents (provenance, CITES certs).
3. Track quota limits if applicable.
